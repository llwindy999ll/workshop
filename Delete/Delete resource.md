
## AWS 리소스 삭제 가이드
 - CASE 1: AWS 계정이 아직 있는 경우
   - [STEP 1](/Delete/STEP1.md) : 비용이 발생한 리소스 확인
      - 계정관리 "Cost Explorer" 로 비용 확인 시 최대 1시간 지연이 있을수 있습니다
   - [STEP 2](/Delete/STEP2.md) : 리소스 관리 콘솔로 이동
